# First-Therapeutic-Efficacy-Study-TES-in-Djibouti
First Therapeutic Efficacy Study (TES) in Djibouti: A Critical Milestone Amid Rising Regional Threats. In Djibouti, where artemether-lumefantrine (AL) has been the first-line treatment since 2014, no prior therapeutic efficacy studies (TES) had assessed in-country drug performance.
